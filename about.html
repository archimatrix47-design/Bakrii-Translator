About

Bakrii Transliterate

What is this?

A transliteration tool for the Sheek Bakrii Saphaloo script — a writing system used for Afar and related Oromo dialects. Type in Latin Qubee, get Sheek Bakrii. Paste Sheek Bakrii, get Latin Qubee. It all happens in your browser. Nothing leaves your device.

Why does it exist?

Most scripts have good digital support. Sheek Bakrii doesn't — yet. The script was proposed for inclusion in the Unicode Standard by Oreen Yousuf and Daniel Yacob, but until it's widely supported, tools like this help people actually use it day-to-day.

This project started as a way to make the script more accessible to researchers, students, and anyone who wants to write in their own language digitally.

How it works

The transliteration engine maps Latin Qubee syllables to their Sheek Bakrii equivalents. It handles consonant-vowel combinations, geminated (doubled) vowels, and preserves spacing and punctuation. The mapping is based on the official character table from the Unicode proposal.

Export uses canvas for PNG output and opentype.js for true vector SVG export — no font installation required on the viewer's end.

Credits

Built on the research and documentation by Oreen Yousuf and Daniel Yacob, whose proposal to encode Sheek Bakrii Saphaloo in the Unicode Standard (L2/24-109) laid the groundwork for digital support of this script.

The typeface used is SheekBakriSaphaloo, designed to render the full character repertoire. Export rendering is powered by opentype.js.
